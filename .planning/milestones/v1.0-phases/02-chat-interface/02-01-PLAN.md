<task type="auto">
  <name>Task 1: Create tenant config module and dynamic route page</name>
  <files>
    src/lib/tenants.ts
    src/app/[tenant]/page.tsx

  <action>
    Create `src/lib/tenants.ts` with a hardcoded tenant map (per project decision — no dynamic routing):

    ```ts
    export type Tenant = {
      slug: string;
      name: string;        // e.g. "Calzedonia AI Assistant"
      displayName: string; // e.g. "Calzedonia"
    };

    export const TENANTS: Record<string, Tenant> = {
      calzedonia: {
        slug: "calzedonia",
        name: "Calzedonia AI Assistant",
        displayName: "Calzedonia",
      },
      sabores: {
        slug: "sabores",
        name: "Sabores AI Assistant",
        displayName: "Sabores",
      },
    };

    export function getTenant(slug: string): Tenant | null {
      return TENANTS[slug] ?? null;
    }
    ```

    Create `src/app/[tenant]/page.tsx` as a Next.js App Router Server Component:
    - Accept `params: { tenant: string }` prop
    - Call `getTenant(params.tenant)` — if null, call `notFound()` from `next/navigation`
    - If found, render a full-height centered layout with:
      - A heading showing `{tenant.name}` (e.g. "Calzedonia AI Assistant")
      - A small subtitle: "Ask me anything about {tenant.displayName}"
    - Use Tailwind for layout: `min-h-screen flex flex-col bg-background`
    - The heading area sits at the top as a header bar with `bg-primary text-primary-foreground p-4`

    This is a Server Component (no "use client" directive needed yet — client interactivity added in Plan 02).
  </action>
  <verify>
    Run `npm run dev` in the project root, then:
    1. `curl -s http://localhost:3000/calzedonia | grep -i "Calzedonia AI Assistant"` — must return a match
    2. `curl -s http://localhost:3000/sabores | grep -i "Sabores AI Assistant"` — must return a match
    3. `npm run build` completes without TypeScript or lint errors
  </verify>
  <done>
    GET /calzedonia returns 200 with "Calzedonia AI Assistant" in the HTML. GET /sabores returns 200 with "Sabores AI Assistant" in the HTML. Build passes cleanly.
  </done>
