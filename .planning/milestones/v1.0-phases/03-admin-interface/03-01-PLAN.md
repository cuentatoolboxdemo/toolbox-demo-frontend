Create `src/components/admin/AdminPage.tsx` as a `"use client"` component. It must implement:

**Auth state:**
- `const [authed, setAuthed] = useState(() => sessionStorage.getItem("adminAuth") === "true")` — SSR-safe: wrap initialization in a lazy initializer that checks `typeof window !== "undefined"` first; use `false` as fallback on server.
  Correct pattern: `useState<boolean>(() => { if (typeof window === "undefined") return false; return sessionStorage.getItem("adminAuth") === "true"; })`
- `const [password, setPassword] = useState("")`
- `const [error, setError] = useState("")`

**Login handler:**
- `function handleLogin(e: React.FormEvent)` — calls `e.preventDefault()`, checks `password === "demo123"`, if correct: `sessionStorage.setItem("adminAuth", "true")`, `setAuthed(true)`, clear error; if wrong: `setError("Invalid password")` (do NOT set authed).

**Logout handler:**
- `function handleLogout()` — `sessionStorage.removeItem("adminAuth")`, `setAuthed(false)`, `setPassword("")`, `setError("")`.

**Login card (rendered when `!authed`):**
- Centered vertically and horizontally: `min-h-screen flex items-center justify-center bg-background`
- A card div: `w-full max-w-sm p-8 rounded-2xl border bg-card shadow-sm`
- Heading: `<h1 className="text-xl font-semibold mb-6">Toolbox Admin</h1>`
- `<form onSubmit={handleLogin} className="flex flex-col gap-4">`
- `<Input type="password" placeholder="Password" value={password} onChange={e => setPassword(e.target.value)} autoFocus />`
- `<Button type="submit" className="w-full">Sign In</Button>`
- `{error && <p className="text-sm text-red-500">{error}</p>}`
