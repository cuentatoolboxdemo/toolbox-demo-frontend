# Phase 7 Plan 1: Centralized Landing Page

## Objective
Create a directory at `/` to index all active tenants, including the new `ide_marketing` tenant, and allow users to navigate directly to their chat interface.

## Steps
1. Add `"ide_marketing": "IDE Marketing Assistant"` to the tenants object in `src/lib/tenants.ts`.
2. Update `src/app/page.tsx` export default page function to read the `tenants` object and display them as clickable cards grouping them in a grid.
3. Each tenant card will have an `href="/[tenant]"` to route the user to their respective conversational UI.

## Verification
- User navigates to `/` and sees `Calzedonia`, `Sabores`, and `IDE Marketing` cards.
- Clicking any card correctly navigates to `/[tenant]`.
