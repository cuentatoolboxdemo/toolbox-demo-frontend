# Phase 5 Plan 2: React Markdown for Citations

## Objective
Enhance the Chat UI to render Assistant responses containing Markdown formatting (bolding, lists, links to source PDF pages).

## Steps

1. **Install Dependency**:
   - Run `npm install react-markdown`.

2. **Update MessageList**:
   - In `src/components/chat/MessageList.tsx`, import `ReactMarkdown` from `react-markdown`.
   - Locate the `<div ...>{message.content}</div>` inside the `map` loop.
   - If `message.role === "assistant"`, replace `{message.content}` with `<ReactMarkdown className="prose prose-sm dark:prose-invert break-words">{message.content}</ReactMarkdown>`.
   - Note: Since we use Tailwind, if `@tailwindcss/typography` is not installed, we should either install it or use basic custom CSS classes for the `a`, `p`, `ul`, `li` tags inside the `components` prop of `ReactMarkdown` to ensure lists and links look decent. Given the minimal setup, we can map basic components:
     - `a`: styled with `text-blue-500 underline`
     - `p`: styled with `mb-2 last:mb-0`
     - `ul`: styled with `list-disc pl-4 mb-2`

## Verification
- Open the application chat at `/`.
- Write a hardcoded message simulating markdown: `**Bold text** and a [Link](https://google.com)` (by intercepting the response or sending it to the assistant).
- Verify the UI renders proper bold text and a clickable blue link instead of raw markdown text.
